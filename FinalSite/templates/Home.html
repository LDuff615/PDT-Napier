What is this site?

This site was created by 6 Edinburgh Napier students who were tasked with creating a website that could give PDTs refresher training for their duties as PDTs. It gives each user the chance to login with either a normal account or an admin account. The normal account can go the home page, training page and quiz page. On the home page the user is given the ability to go to the other sections of the site whilst also being able to logout, learn about the site and PDTs and go to various sections throughout the site and PDT Toolkit site. The training page allows the user to view what quizzes they have completed and then also go to one of the three quiz options. The quiz page allows the user to carry out whatever quiz they have chosen and then be told whether they got a question right or wrong, if they got it wrong then they are given a link to the PDT Toolkit which they can go to for more information. An admin user can do all the same things as what a guest user can do except that they can also go to the admin controls. On the admin controls, the user is taken to the CMS (Content Managment System) which allows them to edit, add and delete questions for the quizzes and allows general editing of the database, there is also instructions on how to use the CMS.
